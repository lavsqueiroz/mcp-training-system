# 🚀 MCP Training System

> Plataforma interativa de treinamento para construção de MCP (Model Context Protocol) Servers

## 📋 Sobre o Projeto

O MCP Training System é uma plataforma web educacional que ensina desenvolvedores a criar integrações poderosas com LLMs através do Model Context Protocol. 

### ✨ Características Principais

- 🎓 **Módulos Progressivos**: Aprenda do básico ao avançado
- 💻 **Editor Integrado**: Escreva e teste código diretamente no navegador
- 🧪 **Sandbox de Testes**: Ambiente seguro para experimentação
- 📊 **Sistema de Progresso**: Acompanhe seu aprendizado
- 🏆 **Gamificação**: Badges e certificações
- 🌐 **100% Web**: Sem instalação necessária

## 🛠️ Stack Tecnológica

- **Frontend**: Next.js 14 + React 18
- **Styling**: Tailwind CSS
- **Editor**: Monaco Editor (VS Code)
- **State**: Zustand
- **Backend**: Next.js API Routes

## 📚 Módulos de Aprendizado

### Módulo 1: Fundamentos

### Módulo 4: Avançado
- Performance e otimização
- Segurança
- Deploy em produção

## 🚀 Getting Started

```bash
# Clone o repositório
git clone https://github.com/lavsqueiroz/mcp-training-system.git

# Instale as dependências
cd mcp-training-system
npm install

# Execute o servidor de desenvolvimento
npm run dev
```
